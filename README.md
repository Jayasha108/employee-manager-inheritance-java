# employee-manager-inheritance-java

This project demonstrates **Inheritance in Java** using an example of employees and managers.

## Problem Statement
A company wants to manage different types of employees.

1. Create a base class **Employee**

   * Variables: `name`, `salary`
   * Method: `displayDetails()`

2. Create a subclass **Manager**

   * Additional variable: `department`
   * Method: `manageTeam()`

3. Create an object of `Manager` and display:

   * Name
   * Salary
   * Department

## Concepts Used

* Java Classes
* Inheritance
* Constructors
* Method calling
* Object creation

## Program Structure

Employee → Manager

## Sample Output

Employee Name: Krishna
Salary: 75000
Department: IT
Managing the IT department.
